Powered by DUST

Midnight uses a unique fee token called DUST. Generated from NIGHT tokens, DUST grows in value over time and is used exclusively for transaction fees. Unlike traditional gas, DUST cannot be transferred - only spent on fees.

❄️

NIGHT Token

NIGHT is Midnight's native utility token distributed via the Glacier Drop program. NIGHT generates DUST (fee resource) over time - like solar panels generating electricity. Hold NIGHT to power transactions without volatile gas fees.

About Midnight Epochs

Midnight uses 30-minute epochs. Each epoch produces ~50 blocks. The validator committee is set at the start of each epoch. Midnight epochs are anchored to Cardano epochs for cross-chain security.
