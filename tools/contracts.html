About Midnight Smart Contracts

Midnight smart contracts are written in Compact (now called Minokawa), a TypeScript-based domain-specific language designed for privacy-preserving computation using zero-knowledge proofs.

As of April 2026, 105 contracts have been deployed on mainnet since the network launched on March 31, 2026. These contracts range from token transfers and DeFi protocols to privacy-preserving identity and governance applications.

Compact/Minokawa compiles to zero-knowledge circuits that run inside the Midnight virtual machine, enabling developers to build applications where sensitive data stays private while still being verifiable on-chain.
