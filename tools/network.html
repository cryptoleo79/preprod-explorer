About the Midnight Network

Midnight targets 6-second block times, providing fast finality for privacy-preserving transactions. The network launched on March 31, 2026 with a federated validator set.

The Cardano-Midnight bridge enables cross-chain transfers between the two networks. Bridge operations include token shielding/unshielding and state commitments posted to Cardano.

This dashboard auto-refreshes every 30 seconds. All data is sourced live from the NightForge Explorer API.
