Midnight Validator Roadmap

Midnight currently operates with federated validators run by established partners including Google Cloud, Vodafone, MoneyGram, eToro, Blockdaemon, Cardano Foundation, and others. These validators produce blocks and secure the network during the early phases.

In Q2 2026 (Mohalu phase), the network will transition toward decentralization. Cardano Stake Pool Operators (SPOs) will be able to join as Midnight validators, leveraging their existing infrastructure. This will significantly expand the validator set and move Midnight toward a fully permissionless consensus model.

The bridge between Cardano and Midnight enables SPOs to participate in both ecosystems, earning rewards while helping secure Midnight's privacy-preserving blockchain.
