How to Embed

NightForge widgets are free SVG badges that auto-update with live data from the Midnight Network. No API key or authentication required.

  • Copy the HTML <img> tag to embed in any website
  • Copy the Markdown snippet for GitHub READMEs, docs, or wikis
  • Badges refresh automatically on each page load
  • Lightweight SVG format works everywhere images are supported
  • Responsive — badges scale cleanly at any size

Widgets are served from the NightForge Explorer API. Simply point your <img> tag at the endpoint and the badge renders the latest value.
